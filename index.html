Etsy Research App (Interactive Demo)

For Etsy API review: This is a single-user, read-only market research tool used to plan my own Etsy shop before launch. It uses only the official Etsy Open API v3 (e.g., GET /v3/application/listings/active) to analyze active listings and compute aggregated metrics (competition, price bands, shop concentration, freshness). No scraping or Shop Manager automation. Marketplace Insights volume is entered manually. Rate-limit compliant (caching/throttling/backoff). No buyer data.
