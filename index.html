Public App Info

Etsy Market Research Tool

This page provides public information for Etsy API access review. The app is a single-user market research dashboard used to analyze keywords, demand, competition snapshots, pricing signals, and historical trends.

  • No scraping is used.
  • Marketplace Insights data is entered manually by the user.
  • Automated data retrieval uses Etsy Open API only.
  • The tool is for research and decision support before shop launch.
